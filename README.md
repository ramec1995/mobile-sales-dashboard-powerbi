# Mobile Sales Dashboard | Power BI

## Project Overview

Developed an interactive mobile sales dashboard using Power BI to analyze sales performance, customer behavior, transactions, and product trends.

## Tools Used

* Power BI
* Power Query
* DAX

## Key Analysis

* Monthly sales trends
* City-wise sales distribution
* Customer ratings
* Mobile model performance
* Payment method analysis

## Features

* KPI Cards
* Slicers
* Interactive Filters
* Maps
* Charts and Visualizations

## Business Insights

Generated insights to identify top-performing products, cities, payment methods, and sales periods.
